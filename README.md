# Machine Learning Aplicado a las Finanzas 


## 📚 Descripción del curso 

Este curso electivo desarrolla capacidades avanzadas para diseñar, entrenar, validar e interpretar modelos de Machine Learning aplicados a problemas financieros (inversión, riesgo, crédito, FX, volatilidad, detección de regímenes). A diferencia de un curso introductorio, el foco está en que los modelos generalicen en datos financieros, que suelen ser no estacionarios, con dependencias temporales, cambios de régimen y alto riesgo de data leakage y sobreoptimización.

El curso enfatiza la construcción de un pipeline completo: definición del problema, ingeniería de variables, validación correcta para series temporales (walk-forward / time series CV), selección y ajuste de hiperparámetros, interpretación y estabilidad (SHAP, drift), y evaluación orientada a decisión (costos, drawdowns, turnover). También se introducen modelos de alto desempeño (ensembles y deep learning) y nociones de model risk / gobernanza para una aplicación responsable en entornos financieros.

## 🎯 Objetivos del curso

Que el estudiante sea capaz de construir y defender un modelo de ML aplicado a finanzas que sea reproducible, validado correctamente y útil para la toma de decisiones, comprendiendo sus limitaciones y riesgos. Al finalizar el curso, el estudiante podrá:

* Formular problemas financieros como tareas de ML (predicción, clasificación, ranking, detección de regímenes), definiendo targets y horizontes sin incurrir en leakage.

* Diseñar un pipeline reproducible de preparación de datos e ingeniería de variables para series financieras (lags, rolling features, calendarios, etiquetado por horizonte).

* Aplicar esquemas de validación adecuados a finanzas (walk-forward, expanding/rolling windows, TS-CV) y seleccionar métricas consistentes con el objetivo económico.

* Entrenar, comparar y tunear modelos supervisados (lineales, Random Forest, Gradient Boosting), incluyendo manejo de desbalance y calibración cuando corresponda.

* Usar interpretabilidad (SHAP/feature importance) y análisis de estabilidad temporal para evaluar robustez y riesgo de degradación.

* Implementar modelos deep learning (MLP) y secuenciales (RNN/LSTM/GRU) y compararlos críticamente frente a alternativas clásicas (boosting).
	
* Conectar predicciones con decisiones financieras (señales/portafolio) mediante backtesting con costos y métricas de riesgo.

* Elaborar documentación y un plan básico de operación del modelo (versionado, monitoreo de drift, recalibración) con criterios de ética y gobernanza.
